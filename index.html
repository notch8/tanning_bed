Tanning Bed by notch8

Tanning Bed is a generic Solr interface for models, because models are too sexy to get the their tans from the sun directly ;-)

Tanning Bed is Solr for models. Tanning Bed provides a Ruby interface for the Solr (http://lucene.apache.org/solr/) search engine to use in you're models not matter whether they are Datamapper, Active Record, Couchrest or just general Ruby classes.

Dependencies

* Solr (bundled) * Solr-Ruby library (bundled)

Install

sudo gem install notch8-tanning-bed --source http://gems.github.com
